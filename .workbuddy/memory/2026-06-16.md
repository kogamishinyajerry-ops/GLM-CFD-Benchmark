## 关键技术约定（写代码时遵守）
- run_id 格式: `YYYYMMDDTHHMMSSZ_<case_id>_<solver>_<hash8>`（secrets.token_hex(4)）
- `SolverAdapter.prepare(self, case, case_dir, run_dir)` 三参签名（避免污染 CaseSpec）
- 时间戳统一 `datetime.now(timezone.utc).isoformat()`
- 失败/timeout 必须写 manifest（status=failed/timeout + error）
- 所有路径 `pathlib.Path`，禁用 `os.path.join`
- Windows subprocess 必须 `encoding="utf-8", errors="replace"`
